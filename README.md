# Платформа ИИ-агентов

Платформа для развертывания автономных ИИ-агентов, способных выполнять различные бизнес-задачи, такие как создание контент-планов, генерация воронок продаж и аналитика.

## Основные технологии

* **Фреймворк:** FastAPI (Python 3.11+)
* **База данных:** PostgreSQL, SQLAlchemy (ORM), Alembic (миграции)
* **Фоновые задачи:** Celery, Redis, APScheduler
* **ИИ интеграции:** OpenAI (GPT-4), Anthropic (Claude), Whisper
* **Интерфейс:** Telegram Bot API (python-telegram-bot)
* **Контейнеризация:** Docker, Docker Compose

## Структура проекта

* `app/` — исходный код приложения
    * `models/` — модели базы данных SQLAlchemy (пользователи, контент-планы, сессии)
    * `routers/` — эндпоинты API
    * `services/` — бизнес-логика (интеграция с ИИ, Telegram-бот, транскрибация)
    * `prompts/` — шаблоны промптов для генерации ответов ИИ
    * `utils/` — вспомогательные функции
* `migrations/` — файлы миграций базы данных
* `docker-compose.yml` — конфигурация сервисов Docker

## Установка и запуск

1.  **Склонируйте репозиторий:**
    ```bash
    git clone <URL_репозитория>
    cd <название_папки>
    ```

2.  **Настройте переменные окружения:**
    Создайте файл `.env` в корневой директории и укажите необходимые ключи API и настройки подключения:
    ```env
    DATABASE_URL=postgresql://postgres:postgres@db:5432/ai_agents
    REDIS_URL=redis://redis:6379/0
    OPENAI_API_KEY=ваш_ключ_openai
    ANTHROPIC_API_KEY=ваш_ключ_anthropic
    TELEGRAM_BOT_TOKEN=ваш_токен_telegram_бота
    DEBUG=False
    ```

3.  **Запустите приложение:**
    Убедитесь, что у вас установлен Docker, и выполните команду:
    ```bash
    docker-compose up --build
    ```

После запуска:
* API будет доступно по адресу: `http://localhost:8000`
* База данных PostgreSQL будет работать на порту `5432`
* Redis будет доступен на порту `6379`
* Фоновые воркеры Celery запустятся автоматически для обработки тяжелых задач.
